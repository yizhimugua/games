# 五条悟（眼罩）Codex Pet

将此目录放到 Codex 的 `~/.codex/pets/blindfold-gojo/` 后即可作为自定义宠物使用。

- `spritesheet.webp`：8×9 动画图集，1536×1872，透明背景。
- `pet.json`：当前 Codex 支持的宠物清单。
- `activation-katsusa.mp3`：日语男声「勝つさ」（会赢的）。当前 Codex 宠物清单尚无激活音效字段，因此该文件作为配套素材交付，应用不会自动播放。
- `qa/`：联系表、逐行动画预览和验证结果。

动作映射：

- `waving`：鼠标接触时触发的张臂抬膝奔跑动作
- `running`：Codex 工作时触发的托腮、抱臂思考动作
- `waiting`：术式反转「赫」
- `review`：虚式「茈」
